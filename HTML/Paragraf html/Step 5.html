PROFIL SAYA

Nama saya Indra Wahyudi, saya lahir di Bandung 18 Agustus 2001, saya terlahir sebagai anak ke 2 dari saudara saya yang bernama Heni Restiani. Dulu saya ketika di Bandung Broken Home karena ibu dan bapak saya bekerja dan kakak saya ada di kampung di nenek saya.

Walaupun dulu saya Broken Home tapi saya tidak seperti anak-anak pada umumnya yang memanfaatkan hal tersebut untuk main bebas keluar rumah. Tapi waktu itu saya jadikan untuk merelaksasikan diri untuk menenangkan hati saya karena menurut saya sendiri itu lebih tenang dari pada hidup dengan banyak orang.

Walaupun orang tua saya pergi bekerja dan meninggalkan saya sendirian dirumah tapi saya tidak merasa bahwa orang tua saya benci pada saya, justru mereka sangat menyayangi saya karena mereka tidak mau anaknya mati kelaparan. Hari-hari saya itu sangat membosankan kalau boleh jujur karena saya tidak punya teman karena saya terlalu banyak diam dirumah.


Nah itu sedikit riwayat hidup saya yang singkat.
